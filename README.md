# 🐞 Bug Tracker

Bug Tracker  is a software tool designed to help teams to report, track, assign, resolve, and document bugs (errors or issues) in software development and ensure that no bug is forgotten and that every problem is solved systematically.


## 🔏 Roles

.  👤 Admin

.  🧑‍💼 Manager

.  🧑‍💻 Developer

.  🧪 Tester


## 👤 Admin Features

.  👥 Manage Users includes add, update, delete 

.  🐞 Manages Projects, bugs and can update bug status 

.  🗨️ Comment on bugs

.  🔍 Filter users by username , Filter projects by project title, filter bugs by bug title


## 🧑‍💼 Manager Features 

.  🐞 Manages bugs, can update bug status 

.  🗨️ Comment on bugs

.  🔍 Filter bugs by bug title

.  🔁 Assign bugs to developers


## 🧑‍💻 Developer Features 

.  🐞 Rectifies bugs and can update bug status

.  🗨️ Comment on bugs

.  🔍 Filter bugs by bug title


## 🧪 Developer Features 

.  🐞 Raise bugs , can update bugs and can delete

.  🗨️ Comment on bugs

.  🔍 Filter bugs by bug title


## 🛠️ Tech Stack

.  Frontend: HTML, CSS, JS 

.  Backend: Spring Boot, Java

.  Database: MySql

.  Authentication: JWT
